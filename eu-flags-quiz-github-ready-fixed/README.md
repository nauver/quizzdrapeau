# Quiz Drapeaux UE – GitHub ready

Version prête pour **GitHub Pages**.

## Déploiement GitHub Pages
1. Dézipper l’archive
2. Créer un dépôt GitHub
3. Importer tous les fichiers à la racine du dépôt
4. Ouvrir **Settings → Pages**
5. Choisir :
   - **Source** = Deploy from a branch
   - **Branch** = `main`
   - **Folder** = `/ (root)`
6. Attendre la publication
